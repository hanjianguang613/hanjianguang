The usage of anisotropic PP- and PS-wave GB-PSDM is very convenient.
1. The parameters required for the program to run and the corresponding data file path and name are entered through the parameter file canshu.txt.
2. Compile and link the source program, put the relevant data files in the target directory, and it can start to run.
3. The source code pregbmiganwithoutslant.c is used for PP-wave migration, and the source code pspregbmiganwithoutslant.c is for PS-wave.
